# 08-2 再製作形象頁

### 練習重點
```
在整個頁面中自由安排文字及圖片的位置.
```

### 執行結果:
![GitHub Logo](/imgs/results08-2.jpg)


### 檔案放置方式:
```
|__ index.html  
|   
|__ <css>
|     |__ main.css 
|
|__ <imgs>
      |__ photo.jpg   (300像素*450像素) 
```


### (2) main.css

```css
@charset "utf-8";

/*---------------------------*/
/* 編排網頁                   */
/*---------------------------*/
html, body{
    font-family: 'Noto Sans TC', sans-serif; 
    font-size:17px;
    font-weight: 280;   
    background:#000; 
}

/*---------------------------*/
/* 編排<容器>區塊             */
/*---------------------------*/
div.container{
    position:relative;
    width:800px;      
    margin:0px auto 0px auto;   
    background:#000;
}

/*---------------------------*/
/* 編排<內文>區塊             */
/*---------------------------*/
div.content{}

/* 定義照片 */
div.photo{
    position:absolute;
    background:url(../imgs/photo.jpg);
    width:300px;
    height:450px;
    left:20px;
    top:150px;;
}

/* 定義文字 */
div.text{
    position:absolute;
    width:470px;
    left:250px;
    top:210px;
    color:#fff;
    font-size:18px;
    font-weight:300;
    line-height:28px;	
    letter-spacing:1px;
    text-align:right;
}

/* 定義標題 */
div.title{
    position:absolute;
    left:740px;
    top:230px;	
    width:50px;
    height:300px;
    writing-mode:tb-rl;  /*設定為直書, 由右至左*/
    color:#fff;
    font-size:30px;
    line-height:55px;
    letter-spacing:10px;		
}	

/* 定義首字 */
div.text span.large{
    font-size:24px;
}

/* 定義小字 */
div.text span.small{
    font-size:16px;
}

span.red{
    color:#f00;	
}
```
